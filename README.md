# Usage
<!-- usage -->
```sh-session
$ npm install -g bulkup
$ bulkup COMMAND
running command...
$ bulkup (-v|--version|version)
bulkup/0.0.0 win32-x64 node-v12.19.0
$ bulkup --help [COMMAND]
USAGE
  $ bulkup COMMAND
...
```
